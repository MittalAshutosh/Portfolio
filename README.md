# Ashutosh Mittal — Portfolio

A dual-kernel personal portfolio for Ashutosh Mittal: full-stack developer,
technical project manager, and founding-team operator at RSN One.

## The concept

The persistent **BUILD ⇄ RUN** switch re-narrates the same career through two
lenses:

- **BUILD** — architecture, code, integrations, infrastructure, and reliability.
- **RUN** — teams, roadmaps, vendors, delivery, launches, and commercial outcomes.

The active kernel changes the palette, typography, motion voice, experience
copy, project records, capabilities, and contact treatment without remounting
the page.

## Highlights

- Responsive single-page portfolio from 360px to large desktop displays
- BUILD and RUN kernel themes persisted in local storage
- Keyboard controls: `K` switches kernels and `Ctrl/Cmd + K` opens commands
- Expandable project records with active and alternate copy lenses
- Reduced-motion support, semantic landmarks, visible focus states, and AA-safe copy
- Technical and management resume downloads
- Open Graph metadata, social preview, and JSON-LD Person schema
- GSAP and Lenis enhancements with a complete static fallback

## Run locally

Requires Node.js `>=22.13.0`.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Validate

```bash
npm test
```

This creates the production build and verifies the server-rendered portfolio.

## Stack

Vinext, React 19, TypeScript, CSS, GSAP, Lenis, and Cloudflare-compatible Sites
hosting.
